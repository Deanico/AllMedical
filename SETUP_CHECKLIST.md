## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| QR code won't appear | Wait 60 seconds, check black window for errors |
| Phone says "Unable to connect" | Make sure phone and computer on same WiFi |
| Can't login | Use exact same email/password as web dashboard |
| App crashes | Restart START_APP.bat, wait longer |
| Changes don't save | Check internet connection, try again |
| Leads list empty | Pull down to refresh, check web dashboard has leads |
