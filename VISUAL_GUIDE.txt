===========================================
    ALLMEDICAL MOBILE APP - VISUAL GUIDE
===========================================

STEP 1: INSTALL EXPO GO
========================

On Your Phone:
  ____________
 |            |
 | App Store  |  or  Google Play
 |____________|
 
 Search: "Expo Go"
 
 Download (it's FREE!)


STEP 2: RUN THE SQL (ONE TIME)
===============================

Computer Browser:
  _________________________________
 |                                 |
 | supabase.com                    |
 | → Your Project                  |
 | → SQL Editor (left side)        |
 | → New Query                     |
 | → Paste the code                |
 | → Click RUN                     |
 |_________________________________|


STEP 3: START THE APP
======================

On Your Computer:

  File Explorer
    ↓
  AllMedical\mobile folder
    ↓
  Double-click: START_APP.bat
    ↓
  Black window opens
    ↓
  Browser opens with QR code
  
   ████████████████
   ████  ██  ██████
   ████████  ██████   ← Scan this!
   ██  ██████  ████
   ████████████████


STEP 4: SCAN QR CODE
====================

On Your Phone:

 Open Expo Go App
    ↓
 Tap "Scan QR Code"
    ↓
 Point camera at QR code
    ↓
 Wait 10-30 seconds
    ↓
 App loads!

DONE! 🎉
=========

Your mobile app is running!

To stop: Press Ctrl+C in the black window

To start again: Double-click START_APP.bat


TROUBLESHOOTING
===============

Problem: QR code doesn't show
Solution: Wait 60 seconds

Problem: Phone can't connect
Solution: Same WiFi on phone + computer

Problem: Can't login
Solution: Same password as web dashboard

Problem: No leads showing
Solution: Pull down to refresh
